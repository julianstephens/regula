# Assessments & Reviews

Assessments track evaluations, while reviews use spaced repetition to reinforce learning. Learn how to manage both effectively.

## Assessments Overview

An **Assessment** is any type of evaluation: tests, quizzes, projects, papers, presentations, etc.

### Assessment Properties

- **Title**: Name of the assessment (required)
- **Program**: Associated program (required)
- **Module**: Optional module grouping
- **Lesson**: Optional lesson association
- **Due Date**: When it's due or occurs (required)
- **Type**: Kind of assessment (test, project, etc.) (optional)
- **Status**: Current progress state
- **Notes**: Details about the assessment (optional)

### Assessment Status

| Status          | Meaning                       |
| --------------- | ----------------------------- |
| **Not Started** | Haven't begun preparation     |
| **In Progress** | Actively working on it        |
| **Submitted**   | Turned in for grading         |
| **Graded**      | Received results and feedback |
| **Archived**    | No longer active              |

## Creating Assessments

### Via Assessments Page

1. Go to **Assessments** in the sidebar
2. Click **New Assessment**
3. Fill in:
   - **Title**: Name (e.g., "Midterm Exam")
   - **Program**: Which program (required)
   - **Due Date**: When it happens (required)
   - **Type**: What kind (optional)
   - **Status**: Current state
   - **Notes**: Details (optional)
4. Click **Create**

### Via Lesson Detail

1. Open a Lesson
2. Look for **Associated Assessments**
3. Create assessment from there
4. Lesson is automatically linked

### Best Practices

- Create assessments for all graded work
- Include details in notes (weight, rubric, etc.)
- Link to relevant lessons if applicable
- Set due dates aligned with lesson completion

## Managing Assessments

### Filtering

On the **Assessments** page:

**By Status:**

- Select a status to filter
- Shows only assessments in that state
- Leave blank to see all

**By Program:**

- Select a program to filter
- Shows only assessments in that program
- Leave blank to see all

**Combining filters:**

- Select both status and program
- See only matching assessments

### Viewing Assessment Detail

Click an assessment to see:

- Full details (title, dates, notes)
- Associated lesson (if any)
- Edit/Delete buttons
- Status change options

### Editing Assessments

1. Click on the assessment
2. Click **Edit**
3. Update fields:
   - Title
   - Program/Module/Lesson associations
   - Due date
   - Type
   - Status
   - Notes
4. Click **Update**

### Changing Status

Track progress through the assessment workflow:

1. Create → **Not Started**
2. Begin work → **In Progress**
3. Submit → **Submitted**
4. Get results → **Graded**
5. Archive when done → **Archived**

Click **Update Status** to change at any time.

### Deleting Assessments

**Removes the assessment permanently.**

1. Open the assessment
2. Click **Delete**
3. Confirm

Associated lessons aren't deleted, only the assessment.

## Reviews Overview

A **Review** is a spaced-repetition study session. After completing a lesson, reviews appear at increasing intervals to reinforce knowledge.

### How Reviews Work

1. **Complete a Lesson** - Mark lesson as completed
2. **Review Created** - System creates first review (appears in "Reviews Needed")
3. **Log Review** - You study the lesson again
4. **Next Review Scheduled** - System schedules next review
5. **Repeat** - Continue reviews at increasing intervals

### Review Properties

- **Lesson**: Which lesson being reviewed (required)
- **Due Date**: When the review is scheduled
- **Status**: Not started, in progress, or completed
- **Difficulty**: Optional rating
- **Notes**: Optional study notes

### Review Status

| Status          | Meaning                  |
| --------------- | ------------------------ |
| **Not Started** | Due but not yet reviewed |
| **In Progress** | Currently reviewing      |
| **Completed**   | Reviewed and logged      |

## Managing Reviews

### Finding Reviews

Reviews appear in several places:

1. **Dashboard** - "Reviews Needed" section shows due reviews
2. **Lesson Detail** - "Reviews" tab shows all reviews for that lesson
3. **Reviews Page** - All reviews across all lessons (go to **Reviews**)

### Reviewing a Lesson

#### Quick Method (Dashboard)

1. Go to **Dashboard**
2. Find **Reviews Needed** section
3. Click on a review
4. Read through the lesson content
5. Log the review session

#### Full Method (Reviews Page)

1. Go to **Reviews**
2. Click on a review
3. View the associated lesson
4. Study the material
5. Log the review
6. Optionally rate difficulty

### Logging a Review

When actively reviewing:

1. Open the review
2. Click **Start Review** or similar
3. Study the lesson content
4. Use the timer to track time
5. Click **Complete Review**
6. Optionally add notes or difficulty rating

**What happens next:**

- Review is marked completed
- Next review is scheduled at a longer interval
- Original lesson remains marked completed

### Review Intervals

Spaced repetition typically uses intervals like:

- First review: 1 day after completion
- Second review: 3 days after first
- Third review: 1 week after second
- Fourth review: 2 weeks after third
- And so on...

**Note**: Exact intervals depend on your settings and difficulty ratings.

### Filtering Reviews

On the **Reviews** page, filter by:

- **Program**: Show reviews for specific program
- **Status**: Show due, in-progress, or completed reviews
- **Sort**: Order by due date (earliest first)

**Use case**: See all reviews due this week for a specific program.

## Tips for Assessments & Reviews

### Preparing for Assessments

1. Create assessment entry when exam/project is announced
2. Break into smaller study units via lessons
3. Practice problems before due date
4. If possible, take a practice assessment
5. Log sessions as you prepare

### Using Reviews Effectively

- **Don't skip reviews** - They're critical for retention
- **Review consistently** - Daily or weekly review routine
- **Rate difficulty** - Helps system adapt to your needs
- **Take notes** - Use review session notes to capture insights
- **Space them out** - Many small reviews beat cramming

### Linking Assessments to Lessons

When possible:

1. Create lesson for an assignment
2. Create assessment for the same assignment
3. Link assessment to lesson
4. This consolidates information about the work

### Time Tracking

- Log all study sessions (including reviews)
- See total time invested in lessons
- Assessments help validate if time matches expectations
- Reviews should take less time than original learning

### Before an Assessment

1. Go to **Reports** to see overall progress
2. Check **Dashboard** for slipping items
3. Review all lessons for the assessment
4. Log practice sessions
5. Note questions in assessment notes

## Spaced Repetition Benefits

Why reviews matter:

- **Retention**: Active recall strengthens memory
- **Spacing**: Reviewing at intervals is most effective
- **Long-term**: Information moves from short-term to long-term memory
- **Confidence**: Repeated exposure increases mastery
- **Efficiency**: Small frequent reviews beat cramming
