# Resources

**Resources** are reference materials that support your learning—textbooks, websites, videos, articles, and more. Organize and access them easily.

## What Are Resources?

Resources are links to study materials you reference frequently. Use them to:

- Centralize references
- Categorize by subject area
- Mark favorites for quick access
- Link to lessons and programs

## Creating Resources

### Basic Process

1. Go to **Resources** in the sidebar
2. Click **New Resource**
3. Fill in:
   - **Title**: Resource name (required)
   - **URL**: Link to the resource (required)
   - **Type**: Category (textbook, video, website, etc.) (optional)
   - **Area**: Subject area (optional)
   - **Program**: Associated program (optional)
   - **Description**: Notes about the resource (optional)
4. Click **Create**

### Resource Information

- **Title**: Clear name for the resource
  - Examples: "Python Documentation", "Khan Academy Calculus"
- **URL**: Web address to access the resource
  - Examples: "<https://docs.python.org>", "<https://www.khanacademy.org>"
- **Type** (optional): Category
  - Types: "textbook", "video", "website", "article", "tool", etc.
- **Area** (optional): Subject categorization
  - Examples: "Computer Science", "Mathematics", "Languages"
- **Program** (optional): Associated program
  - Link to specific course using the resource
- **Description** (optional): Notes or details
  - Why you use it, what to look for, important sections

## Managing Resources

### Filtering Resources

On the **Resources** page:

**By Area:**

1. Select an area from the dropdown
2. See only resources in that area
3. Select "All areas" to see everything

**By Type:**

- Resources are grouped or can be sorted by type
- Helps find specific material styles

**Search / Sort:**

- Resources are typically sorted alphabetically
- Use area filter for organized browsing

### Viewing Resource Details

Click a resource title to see:

- Full URL (clickable link)
- Type and area
- Associated program
- Description/notes
- Edit/Delete buttons

Click the **URL** to open the resource in a new tab.

### Editing Resources

1. Click on the resource
2. Click **Edit**
3. Update fields:
   - Title
   - URL
   - Type
   - Area
   - Program
   - Description
4. Click **Update**

### Deleting Resources

**Warning**: Permanent action.

1. Open the resource
2. Click **Delete**
3. Confirm

This removes the resource record but doesn't affect linked lessons.

## Organizing Resources by Area

### Using Areas

Track resources by subject:

1. Create resources and assign to **Area**
2. Create multiple resources in same area
3. Filter by area on Resources page
4. Easy to switch between subjects

### Area Examples

- Computer Science
- Mathematics
- Physics
- Languages
- History
- Literature
- Professional Development

### Setting Up Areas

Areas aren't pre-created. Instead:

1. Create resources and mention areas
2. Use consistent area names
3. Other resources use same area names automatically
4. Filter brings similar resources together

## Linking Resources to Programs

When creating a resource:

1. Set the **Program** field
2. Resource is associated with that program
3. View on program detail page
4. Easy access when working in that program

### Viewing Program Resources

1. Open a **Program**
2. Look for associated resources
3. Quick links to materials for that program

## Resource Tips

### Curating Your Resources

- Link to materials you actually use
- Include multiple formats (video + text)
- Remove outdated links
- Mark most-used as favorites

### Describing Resources

In the description, note:

- **Why you use it**: Main purpose
- **When to use it**: When in your study
- **Key sections**: Important parts
- **Login required**: If credentials are needed
- **Cost**: Free or paid
- **Updates**: How frequently it's updated

### Sharing with Study Groups

If using a group:

1. All members add favorite resources
2. Filter by area to find group's best materials
3. Discuss what resources are most valuable

### Maintaining Your List

Regularly:

1. Review resources
2. Remove broken/outdated links
3. Update descriptions with new insights
4. Add new discoveries
