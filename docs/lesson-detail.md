# Lesson Detail Guide

The **Lesson Detail** page shows complete information about a single lesson and all its associated data.

## Accessing Lesson Detail

1. From **Dashboard**, click any lesson
2. From **Calendar**, click a lesson in the date view
3. From **Modules**, click a lesson
4. From **Programs**, click a lesson in the lessons tab
5. From anywhere with lesson links, click the lesson

## Lesson Overview

The main section shows:

- **Lesson Title**: Name of the lesson
- **Module**: Which module it belongs to
- **Program**: Associated program
- **Due Date**: When it's due
- **Available Date**: When you can start
- **Lesson Type**: Category (if assigned)
- **Course**: Associated course (if linked)
- **Status**: Current progress state
- **Content**: Any notes or resource links

### Status Indicators

Visual badge shows current status:

- **Not Started**: Gray/default
- **Active**: In-progress indicator
- **Blocked**: Alert/blocked indicator
- **Completed**: Checkmark/completed indicator

## Lesson Tabs

### Overview Tab

Main information about the lesson:

- All fields listed above
- Edit button to modify
- Links to related items

### Sessions Tab

Study sessions logged for this lesson:

**Shows:**

- Date of session
- Duration in minutes
- Optional session notes
- Total time spent

**Log a new session:**

1. Click **Start Timer**
   - Built-in timer for live study
   - Click Stop when done
   - Choose to save
2. Or click **Log Session**
   - Manually enter time
   - Enter date (default: today)
   - Add optional notes

**Why log sessions:**

- Track time investment
- See total effort
- Reports use session data
- Personal accountability

### Events Tab

Timeline of all changes to this lesson:

**Shows:**

- Status changes (when marked complete, etc.)
- Date modifications
- Other significant events
- Chronological order
- Timestamps

**Use it to:**

- See when you started working
- Understand progression timeline
- Track when things got blocked
- Review history of lesson

### Reviews Tab

Spaced-repetition reviews for this lesson:

**Shows:**

- Review due dates
- Review status
- Difficulty rating (if rated)

**Previous reviews:**

- Shows all past reviews
- Helpful for tracking learning
- Can see review progression

**Start a review:**

1. Click on a due review
2. Study the lesson content
3. Log the review session
4. Rate difficulty (optional)
5. Next review auto-scheduled

## Editing a Lesson

### Opening Edit Mode

1. Click **Edit** button on lesson
2. Form appears with current values
3. Modify any fields

### Modifiable Fields

- **Title**: Lesson name
- **Program**: Associated program
- **Module**: Associated module
- **Due Date**: When it's due
- **Available Date**: When you can start
- **Type**: Lesson categorization
- **Course**: Associated course
- **Content**: Notes and resource links
- **Status**: Current progress state

### Saving Changes

1. Update desired fields
2. Click **Update** or **Save**
3. Changes take effect immediately
4. Dashboard updates accordingly

### Common Edits

**Rescheduling:**

- Click lesson
- Edit Due Date
- Save
- Dashboard shows new due date

**Adding Resources:**

- Open Edit
- Add links to Content field
- Format as markdown if helpful
- Save

**Changing Module:**

- Edit
- Select different Module
- Save
- Lesson moves to new module

## Changing Lesson Status

### Status Workflow

From the lesson detail:

1. **Not Started** → **Active**
   - When you begin work
   - Click to change status

2. **Active** → **Completed**
   - When you finish
   - Submission complete
   - Click to change status

3. **Active** → **Blocked**
   - If dependent on other work
   - Can't proceed
   - Click to change status

4. **Blocked** → **Not Started/Active**
   - When dependencies meet
   - Click to unblock
   - Click to restart

5. **Completed** → **Active**
   - If need to revise
   - Click to reopen

### Marking Complete

1. Open lesson
2. Click status dropdown
3. Select **Completed**
4. Confirm
5. Lesson marked complete
6. First review is created automatically

## Sessions & Time Tracking

### Using the Timer

1. Open lesson
2. Click **Start Timer**
3. Study session begins
4. Click **Stop** when done
5. Review duration
6. Optional: Add notes
7. Click **Log Session**

### Manual Session Entry

1. Open lesson
2. Click **Log Session**
3. Enter duration (minutes)
4. Select date (default: today)
5. Add optional notes
6. Click **Log**

### Session Notes

In session notes, you might include:

- Key concepts learned
- Difficult areas
- Connections to other lessons
- Questions for later
- Resources used

### Viewing All Sessions

Go to **Sessions** tab:

1. See all logged sessions
2. Sorted by date
3. Total duration at top
4. Click individual session to view note

### Deleting Sessions

If logged incorrectly:

1. Go to Sessions tab
2. Click session
3. Click Delete
4. Total time recalculates

## Working with Content

### Adding Content

Content can include:

- **Links**: To textbooks, videos, articles
- **Notes**: Key concepts, summary
- **Instructions**: How to complete
- **Resources**: Tools or materials needed
- **Questions**: Things to remember
- **Examples**: Relevant examples

### Formatting Content

Content field supports:

- Plain text
- Links (paste URL)
- Lists
- Basic formatting

Example:

```
**Chapter 3: Algorithms**
- Read pages 45-67
- Watch: https://example.com/video
- Solve: Problems 1-10 at end of chapter
- Key concepts: sorting, search, complexity
```

### Updating Content

1. Open lesson
2. Click Edit
3. Modify Content field
4. Save
5. Changes appear immediately

## Deleting a Lesson

### Warning

Deleting is permanent. Before you delete:

- Consider archiving program instead
- Data cannot be recovered
- Sessions and reviews are lost

### To Delete

1. Open lesson
2. Click **Delete** (usually bottom of page)
3. Confirm deletion
4. Lesson permanently removed

## Tips for Lesson Management

### Accurate Completion

Mark complete only when truly done:

- Reading finished, not just started
- Assignment submitted, not in progress
- Concept understood, not just reviewed

### Regular Updates

- Update status as you work
- Add notes to content
- Log sessions consistently
- Modify dates if needed

### Linking Related Content

In content or notes:

- Reference other lessons
- Link to similar concepts
- Build connections
- Create knowledge web

### Preparation Before Starting

Before opening lesson:

- Check available date
- Read content/links
- Gather materials
- Set timer
- Begin work

### Session Time Management

- Log sessions immediately after
- Use timer for focused work
- Include short breaks in total
- Estimate before doing
- Compare estimate vs. actual

## Troubleshooting

### Status Won't Change?

- Ensure you have permission
- Try refreshing page
- Check browser console for errors

### Sessions Not Showing?

- May take moment to appear
- Try refreshing page
- Verify session was actually logged

### Content Not Saving?

- Check for very long content
- Try breaking into separate lines
- Check browser limits
