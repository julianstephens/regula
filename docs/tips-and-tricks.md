# Tips & Tricks

Make the most of Regula with these best practices and strategies.

## Dashboard Mastery

### Establish a Daily Routine

1. **Morning**: Check Dashboard for today's work
2. **Morning**: Note "Slipping" items needing immediate attention
3. **Throughout day**: Check "In Progress" to resume work
4. **End of day**: Log sessions and mark complete
5. **Before bed**: Check tomorrow's "Due Today"

### Use Block Progress to Motivate

- Watch the progress bar grow as you complete lessons
- Completion % increases with each lesson finished
- Small wins add up to program completion
- Celebrate milestones!

### Prioritize Urgent Items

From Dashboard:

1. Address "Slipping" items first
2. Then "Due Today" items
3. Then "In Progress" items
4. Finally "Reviews Needed"

This order ensures nothing falls through cracks.

### Schedule Reviews Strategically

- Don't let reviews pile up
- 10-15 minute reviews 2-3x per week
- Better than cramming all at once
- Spaced repetition works best when spread out

## Program & Module Planning

### Start Big, Then Break Down

1. Create full-year program with all dates
2. Add term-level programs within year
3. Add block-level within terms
4. Add modules within blocks
5. Add lessons within modules

This top-down approach ensures alignment.

### Balance Your Workload

1. Before creating all lessons, estimate total weeks
2. Spread lessons evenly across this time
3. Leave 10-15% buffer time
4. Check calendar after adding—should look balanced

Example:

- 16-week term
- ~80 lessons total
- ~5 per week average
- Some weeks lighter, others heavier

### Use Modules for Weekly Planning

Create one module per week:

- Module title: "Week 3: Chapters 5-6"
- Module dates: Mon-Sun of that week
- Lessons inside: Daily assignments
- Provides natural cadence

### Name Clearly

For easy filtering:

- Module: "Week 3: Chapters 5-6" (not just "Week 3")
- Lesson: "Chapter 5 Reading - page 1-40" (descriptive)
- Program: "Spring 2026 CS101" (not just "CS101")

## Lesson Management

### Log Sessions Immediately

After studying:

1. While still at desk, log the session
2. Enter time and optional notes
3. Prevents forgetting later
4. Data stays accurate

**Why it matters:**

- Accurate time tracking
- Reports reflect reality
- Better insights into learning pace
- Personal accountability

### Use Lesson Types Strategically

Categorize with types:

- Helps you understand content diversity
- Filter by type to practice specific skills
- Assessments stand out as high-stakes
- Readings vs. problems vs. practice

### Include Resource Links

In lesson content:

1. Add direct link to chapter
2. Link to textbook if online
3. Link to lecture video
4. Link to any needed materials

**Format example:**

```
- Reading: Chapter 3 (pages 45-67)
- Video: https://example.com/lecture-3
- Problems: Section 3.5, problems 1-20
```

### Mark Complete Immediately

When done with lesson:

1. Don't wait until end of day
2. Mark "Completed" right away
3. This triggers review creation
4. Dashboard updates accurately
5. Slipping counter adjusts

## Time Tracking

### Be Honest with Logging

Accurate data helps:

1. Log ALL study sessions
2. Include reading, problems, studying
3. Include office hours attendance
4. Include study group time
5. Even rough estimates help

### Calculate Study Time

Use sessions to estimate:

- Total hours per lesson
- Average time per lesson type
- Hours per week comfortable
- Which topics take longer

Example insights:

- "Problem sets take 2x reading time"
- "I study 18 hours per week"
- "Reviews take 1/3 original study time"

### Set Weekly Study Goals

Based on logging data:

1. Calculate average weekly hours
2. Set sustainable target (e.g., 15 hours)
3. Monitor in Reports weekly
4. Adjust if consistently over/under
5. Track toward goal

## Assessment Strategy

### Create Assessment Entries Early

When assignment announced:

1. Create assessment entry immediately
2. Include weight/importance
3. Note exam type (essay, multiple choice, etc.)
4. Link to relevant lessons
5. Set realistic due date

### Prepare Early

For assessments:

1. Space study sessions before due date
2. Don't cram—use spaced practice
3. Log prep sessions to track effort
4. Use Reviews to reinforce
5. Take practice tests if available

### Track Assessment Time

From Reports:

1. See how long assessments take
2. Use patterns for future planning
3. "Exams usually take 12 hours prep"
4. "Projects take 8 hours"
5. Plan accordingly next term

## Review Excellence

### Maintain Review Discipline

- Don't skip reviews
- Do reviews consistently
- Better 10 minutes daily than 2 hours once weekly
- Consistency beats intensity

### Create Review Routine

Example routine:

- Monday morning: Review 2-3 items
- Wednesday evening: Review 2-3 items
- Friday evening: Review remaining
- Spreads across week naturally

### Use Difficulty Rating

When completing reviews:

1. Rate difficulty: Easy, Medium, Hard
2. Hard reviews appear sooner
3. Easy reviews spaced further
4. System adapts to your needs

### Build on Knowledge

During reviews:

1. Note new insights in review notes
2. Connect to other lessons
3. Look for patterns
4. Build deeper understanding
5. This converts short-term memory to long-term

## Calendar Navigation

### Plan Semester at a Glance

1. Go to Calendar
2. Click through all months of term
3. Look for workload patterns
4. Make adjustments early
5. Verify it's sustainable

### Spot Overload Weeks

In calendar:

1. Dates with 5+ lessons are intense
2. Identify these early
3. Redistribute lessons to adjacent weeks
4. Prevent burnout

### Track Momentum

As weeks pass:

1. Return to calendar
2. Click past dates to see completion
3. Watch as lessons turn "completed" status
4. Motivating to see progress

## Filtering for Focus

### By Program

Focus on one program:

1. Across all pages: Filter to one program
2. Dashboard: Shows only that program block
3. Modules: Shows only that program's modules
4. Reviews: Shows only that program's reviews
5. Reduces cognitive load

**Use case:** When switching between 2-3 programs.

### By Status

Find specific items:

1. Assessments page: Show "In Progress" only
2. Lessons page: Show "Blocked" only
3. Reviews page: Show "Not Started" only
4. Helps prioritize

### By Time

Focus on current period:

1. Dashboard "Ahead Weeks": Only see near future
2. Reports this week/month: Current performance
3. Calendar current month: Daily view
4. Prevents overwhelm from distant future

## Avoiding Common Pitfalls

### Don't Overcommit

- Use calendar to validate schedule
- If consistently overloaded, reduce lessons
- Better to complete less fully than fail at more

### Don't Ignore Slipping

- Slipping items need immediate action
- Unblock them or catch up
- Prevents cascading failures
- Address immediately upon seeing

### Don't Skip Logging

- Data is only useful if complete
- Missing sessions = inaccurate reports
- Invest 1 minute to log right after session
- Saves hours later in analysis

### Don't Let Reviews Pile Up

- Week with no reviews: Great!
- Week with 20 reviews due: Problem!
- Do reviews consistently
- Schedule time for them like lessons

### Don't Mark Complete Too Early

- Mark "Completed" only when truly done
- Submitting doesn't mean mastered
- Dashboard accuracy depends on honesty
- Reviews won't appear until truly completed

### Color Coding Mindset

- Helps mental organization

### Study Group Use

If studying with others:

1. All add same resources
2. Create same program structure
3. Everyone logs sessions

5. Can see group dynamics

5. Helps plan together

### Create Study Journal

4. Creates portfolio of learning
5. Great for future reference

## Performance Optimization

### For Slow Dashboard

1. Go to Settings
2. Uncheck archived programs
3. Only keep active programs visible
4. Dashboard loads faster
5. Still in database, just hidden

### For Cleaner View

1. Archive completed programs
2. Only active programs checked
3. Dashboard shows current work
4. Still can search archived if needed
5. Mental clarity improves

## Building Sustainable Habits

### Start Small

- Don't create 200 lessons first week
- Start with one program/module
- Add more as you get comfortable
- Build habit before scaling

### Log Sessions Daily

- 1-minute daily logging
- 5 minutes creates habit
- Becomes automatic
- Data accumulates naturally

### Weekly Check-in

- 10 minutes every Sunday
- Look at next week in calendar
- Check any slipping items
- Update as-needed
- Stay ahead proactively

### Monthly Review

- 20 minutes monthly
- Check overall reports
- Evaluate pace and balance
- Celebrate completed programs
- Plan next month
