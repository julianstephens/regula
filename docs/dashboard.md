# Dashboard Guide

The **Dashboard** is your main hub for tracking daily study activities and progress. It shows you what to focus on today.

## Getting to the Dashboard

1. Log in to Regula
2. You're automatically on the Dashboard, or click **Dashboard** in the sidebar

## Dashboard Sections

### Block Progress (When Active)

Shows your progress through active study blocks.

**Displays:**

- Block name and weeks (e.g., "Block 1 • Weeks 1-4")
- Progress bar showing completion percentage
- Completion status

**Use it to:**

- Track overall program progress
- See which week you're in
- Monitor completion percentage

**Click** on a block to view its details.

### Due Today

Lessons that must be completed today.

**Displays:**

- Lesson title
- Program it belongs to
- Status badge

**Use it to:**

- Know what needs to be done today
- Quickly jump to a lesson
- See at a glance how many items are due

**Status Indicators:**

- Not started lessons appear in default color
- In progress lessons show their current state
- Completed lessons show completion status

### In Progress

Lessons you've actively started but haven't completed yet.

**Displays:**

- Lesson title
- Program
- Current status

**Use it to:**

- Quickly resume work on active lessons
- See all items you're actively working on

### Slipping

Lessons that are overdue or at risk of becoming overdue soon.

**Indicators of "slipping":**

- Urgency band (color-coded):
  - **Red**: Severely overdue (more than 1 week late)
  - **Orange**: Overdue (up to 1 week late)
  - **Yellow**: At risk (due soon, even if ahead_weeks hasn't reached)

**Use it to:**

- Identify items that need immediate attention
- Prioritize your study time
- Avoid letting things fall too far behind

**Click** on a slipping lesson to update its status or reschedule it.

### Blocked

Lessons you can't start because they depend on other lessons.

**Why items are blocked:**

- Prerequisites not yet completed
- Dependent lessons must finish first
- Limited resources available

**Use it to:**

- Understand dependencies in your study plan
- Know what to prioritize to unblock items
- Plan your work flow

**Tips:**

- Complete prerequisite lessons to unblock dependent ones
- Questions about blockers? Check the program detail view

### Reviews Needed

Lessons completed and now eligible for spaced-repetition reviews.

**Displays:**

- Lesson title
- Program
- When the review is due

**Use it to:**

- Practice spaced repetition
- Reinforce learning
- Keep knowledge fresh

**Review Process:**

1. Click on a review item
2. Review the lesson content
3. Log a review session
4. Regula schedules the next review

### Getting Started Card (New Users)

If you haven't created any study material yet, you'll see a **Getting Started** card with steps:

1. Create a Program
2. Add Modules
3. Add Lessons
4. Start Reviewing

Follow these steps to begin. You can also import a program using the **Import Program** button.

## Dashboard Tips

### Refresh Your Data

The dashboard automatically updates with new information. You can also:

- Refresh the page to get the latest data
- Click on individual sections to refresh them

### Adjust "Ahead Weeks"

Control how far ahead lessons appear:

1. Go to **Settings**
2. Set **Ahead Weeks** (default: 2)
3. Lessons will appear up to (today + ahead_weeks)

### Filter by Block

If viewing multiple active blocks:

- Click on a block to filter the dashboard
- See only lessons for that block
- Reset filter to see all

### Track Your Progress

The dashboard shows:

- Total blocks and their status
- Completion percentages
- Time-to-completion estimates (when available)

### Daily Routine

**Best practices for using the dashboard daily:**

1. Check **Due Today** first
2. Work through those items
3. Check **Slipping** for anything that needs attention
4. Log completed lessons
5. Check **Reviews Needed** for spaced repetition
6. Look at **In Progress** to resume unfinished work

### Performance Summary

At the top of the dashboard (when blocks are active):

- Overall completion percentage
- Week-by-week breakdown for active blocks
- Time remaining in active blocks

## No Programs Yet?

If you haven't created any content:

1. Click **Go to Programs** to create your first program
2. Or click **Import Program** to import from YAML
3. Once created, you'll see study items on the dashboard

## Customization

### Settings Related to Dashboard

Go to **Settings** to adjust:

- **Block Weeks**: Default duration for study blocks (affects program creation)
- **Ahead Weeks**: How far ahead to show available lessons
- **Active Programs**: Toggle which programs appear on dashboard

See [Settings Guide](settings.md) for more details.
