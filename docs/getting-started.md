# Getting Started with Regula

Welcome to Regula! This guide will help you get started with creating and managing your study plans.

## Logging In

1. Navigate to the Regula application
2. Enter your credentials on the login page
3. Click **Sign In**

Once logged in, you'll be taken to the **Dashboard**, your main hub for tracking study progress.

## Your First Steps

Follow these steps to set up your study plan:

### 1. Create a Program

A **Program** is the top-level container for your study plan. Programs can be:

- **Year**: A full academic year
- **Term**: A semester or term (typically 12-16 weeks)
- **Block**: A shorter focused period (2-8 weeks)
- **Custom**: Any other study period

**To create a program:**

1. Go to **Programs** in the sidebar
2. Click **New Program**
3. Enter the program name (e.g., "Spring 2024 CS101")
4. Select the type
5. Set start and end dates
6. (Optional) Assign to an area if you've created one
7. Click **Create**

### 2. Add Modules

A **Module** is a collection of related lessons within a program. Break your program into logical units.

**To create a module:**

1. Go to **Modules** in the sidebar
2. Click **New Module**
3. Enter the module title (e.g., "Chapter 3: Algorithms")
4. Select the program
5. Set start and end dates
6. Click **Create**

### 3. Add Lessons

A **Lesson** is an individual learning unit. Each lesson has a due date and can track your progress.

**To add a lesson:**

1. Open a module or go to **Modules** in the sidebar
2. Click on a module to view its details
3. Click **Add Lesson** or go to a specific lesson
4. Fill in:
   - **Title**: What you're learning
   - **Due Date**: When it's due
   - **Available Date**: When you can start (optional)
   - **Content**: Links to resources or notes (optional)
5. Click **Create**

### 4. Track Your Progress

Return to the **Dashboard** to see:

- Lessons due today
- Active study blocks and their progress
- Lessons that need reviews
- Any blocked or overdue items

## Importing from YAML

Regula supports importing programs and lessons from YAML files. This is useful for importing syllabi or course materials.

**To import a program:**

1. Go to **Programs** in the sidebar
2. Click **Import Program**
3. Provide the YAML course data
4. Click **Import**

See the [YAML Import Guide](yaml-import.md) for details on the file format.
