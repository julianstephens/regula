# Calendar & Planning

The **Calendar** view helps you visualize all lessons by date. Use it to plan your study schedule and see your workload.

## Accessing the Calendar

1. Click **Calendar** in the sidebar
2. The current month is displayed
3. Navigate between months as needed

## Calendar Features

### Month View

- **Grid layout** shows all days of the month
- **Today is highlighted** in the calendar
- **Dates with lessons** show a badge with count
- **Click a date** to see lessons for that day

### Navigation

**Move between months:**

1. Click **←** button to go to previous month
2. Click **→** button to go to next month
3. Current month and year display between buttons

**View current month:**

- Reload the page to reset to today

### Lesson Display

**For each date (in selected day view):**

- Lesson title
- Associated program
- Lesson type (if set)
- Lesson status badge
- Direct link to lesson detail

**Lessons shown:**

- Available dates within the calendar month
- Sorted by available_on date
- Up to your "Ahead Weeks" setting

## Planning Your Study

### Viewing Your Workload

1. Click on several dates to see upcoming lessons
2. Look for dates with many lessons
3. Plan spread-out due dates to balance workload
4. Use calendar to validate lesson scheduling

### Identifying Gaps

- Dates with no lessons indicate free time
- Use gaps for review, projects, or studies not yet entered
- Balance gap size with lesson density

### Planning Obstacles

**Too many lessons on one day?**

- Space due dates differently
- Use modules to organize by week
- Adjust available dates to make lessons appear sooner

**Nothing due next week?**

- Ensure ahead_weeks setting is appropriate
- Check if lessons are marked with correct available_on date
- Consider if scheduling is realistic

## Adjusting Lesson Dates

If using the calendar highlights problems:

1. Identify the lesson with problematic date
2. Click the lesson from the calendar
3. Open lesson detail
4. Click **Edit**
5. Adjust **Due Date** or **Available Date**
6. Click **Update**
7. Return to calendar to verify

## Using Calendar for Planning

### Setting Up a Term

1. **Create Program** - Set term dates
2. **Create Modules** - One per week/unit
3. **Create Lessons** - Add within modules
4. **View Calendar** - See the full schedule
5. **Adjust as needed** - Move dates if overloaded

### Pre-Planning a Course

Before starting:

1. Open calendar
2. Set academic calendar (start/end dates)
3. Create program spanning that time
4. Estimate module count
5. Space modules evenly
6. Create lessons with staggered due dates
7. Verify calendar looks reasonable

### Identifying Crunch Weeks

1. View the calendar
2. Look for dates with 3+ lessons
3. Consider spreading to other dates
4. Balance workload across weeks

## Calendar Settings

### Ahead Weeks

Controls how far in advance lessons appear:

**To adjust:**

1. Go to **Settings**
2. Set **Ahead Weeks** value
3. Click **Save**
4. Return to calendar to see updated range

**Example behaviors:**

- Ahead Weeks: 1 → See 1 week ahead
- Ahead Weeks: 2 → See 2 weeks ahead (default)
- Ahead Weeks: 4 → See 1 month ahead

### Viewing Past Lessons

The calendar shows:

- Current month (all dates)
- Future lessons within ahead_weeks
- Completed lessons (depending on status)

To review past:

1. Navigate to previous month
2. Click past dates
3. See completed lessons
4. View status and times

## Tips for Effective Planning

### Regular Review

- Check calendar weekly
- Identify next week's workload
- Plan your schedule
- Flag any overload situations

### Balance Academic and Personal

- Use calendar to plan around personal time
- Note personal commitments
- Ensure study schedule is sustainable

### Realistic Scheduling

- Factor in non-study time
- Account for breaks and rest days
- Spread lessons across multiple days
- Avoid consecutive heavy-load days

### Using with Dashboard

- **Calendar** shows what's due when
- **Dashboard** shows what needs attention today
- Together they provide full picture
- Plan using calendar, execute using dashboard

### Export Planning

- Consider exporting lessons to external calendar
- Share academic calendar with study group
- Sync with personal calendar app

## Common Planning Scenarios

### Scenario: Too Much Due on Friday

**Problem:** All weekly lessons due Friday
**Solution:**

1. Review lessons in calendar
2. Adjust some due dates to earlier in week
3. Spread across Monday-Thursday
4. Verification: Friday should have fewer items

### Scenario: Week with No Lessons

**Problem:** A week appears empty
**Solution:**

1. Check if lessons exist but aren't available yet
2. Verify available_on dates aren't too far in future
3. Consider if content truly not needed that week
4. Check ahead_weeks isn't limiting view

### Scenario: Too Many Modules at Once

**Problem:** Many modules overlap in dates
**Solution:**

1. Add more weeks to program (extend end date)
2. Shift module start/end dates apart
3. Reduce lessons per module
4. Add intermediate review weeks
