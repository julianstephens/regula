# Settings & Preferences

The **Settings** page lets you customize your Regula experience and manage your data.

## Accessing Settings

1. Click **Settings** in the sidebar
2. View all available options
3. Make changes and save

## General Settings

### Block Weeks

The default duration for study blocks in term-type programs.

**What it does:**

- Divides terms into focused study periods
- Example: 4 weeks per block = term divided into 4-week sections
- Can override per program

**To adjust:**

1. Find **Block Weeks** setting
2. Enter default value (typically 2-8 weeks)
3. Click **Save**

**Use cases:**

- 4-week blocks: Standard academic terms
- 6-week blocks: Longer intensive periods
- 2-week blocks: Very rapid programs

**Note**: Changing this doesn't affect existing programs, only new ones.

### Ahead Weeks

How far in advance Regula shows available lessons.

**What it does:**

- Dashboard shows lessons up to (today + ahead_weeks)
- Calendar displays available lessons within range
- Controls what appears in "available" lessons

**To adjust:**

1. Find **Ahead Weeks** setting
2. Enter number of weeks ahead (default: 2)
3. Click **Save**

**Examples:**

- Ahead Weeks: 1 → See 1 week of upcoming work
- Ahead Weeks: 2 → See 2 weeks ahead (default)
- Ahead Weeks: 4 → See a full month ahead

**Use cases:**

- 1 week: Prefer week-to-week planning
- 2 weeks: Balanced visibility
- 4+ weeks: Like seeing long-term schedule

## Active Programs

Control which programs appear on your dashboard.

**What it does:**

- Only "active" programs show in dashboard blocks
- Reduces clutter if managing many programs
- Mark programs complete/archived separately

**To manage:**

1. Find program list in settings
2. Checkbox next to each program
3. Check = appears in dashboard
4. Uncheck = hidden from dashboard
5. Click **Save**

**Use cases:**

- Hide archived programs
- Focus on current semester only
- Reduce dashboard clutter
- Still in database, just hidden

## Data Management

### Export Data

Export all your study data for backup or analysis.

**Available formats:**

- **JSON**: Computer-friendly, preserves all structure
- **CSV**: Spreadsheet-friendly, can open in Excel

**Collections you can export:**

- Programs
- Modules
- Lessons
- Assessments
- Reviews
- Resources
- Study Sessions
- Item Events
- User Settings

**To export:**

1. Find **Export Data** section
2. Select collection to export
3. Choose format (JSON or CSV)
4. Click **Export**
5. File downloads to your computer

**Use exports for:**

- Backup before major changes
- Analyze in Excel/Sheets
- Share progress with advisors
- Migrate to another system
- Archive completed programs

### Exporting Individual Collections

Each collection has an export button:

- **Programs**: All programs you created
- **Modules**: All modules with structure
- **Lessons**: All lessons with dates/status
- **Assessments**: All assessments with scores
- **Reviews**: All reviews with completion
- **Resources**: All resources with links
- **Study Sessions**: All time logs
- **Item Events**: Complete timeline of changes
- **User Settings**: Your preferences

**CSV files can be:**

- Opened in Excel/Google Sheets
- Analyzed for trends
- Printed for records
- Shared as reports

**JSON files can be:**

- Backed up
- Imported into other tools
- Analyzed programmatically
- Archived with structure intact

## Account Settings

### Session Management

- **Current User**: Shows your login
- **Sign Out**: Logs you out securely
- **Last Login**: When you last accessed Regula

**To sign out:**

1. Go to Settings
2. Click **Sign Out** button
3. Redirected to login page
4. Session is closed

## Saving Changes

After modifying settings:

- **Click Save** to apply changes
- Page will confirm settings saved
- Some changes take effect immediately
- Others apply to new items created

**Changes like:**

- Block Weeks: Apply to new programs
- Ahead Weeks: Apply immediately to dashboard
- Active Programs: Apply immediately
- Export: Downloads file

## Settings Tips

### Organizing Your Workspace

1. Set reasonable ahead_weeks (usually 1-2)
2. Set block_weeks to match your schedule
3. Uncheck completed/archived programs
4. Keep only active programs visible

### Data Backup

Regular backups help:

1. Export all data monthly
2. Save to external drive
3. Or cloud backup location
4. Protects against loss

### Performance

If dashboard feels slow:

- Too many active programs?
- Uncheck some in settings
- Reduces data loaded

### Planning Next Term

Before switching terms:

1. Archive old programs (still there, hidden)
2. Export final data
3. Create new programs for term
4. Reset ahead_weeks if needed

## Advanced Settings

### Clear Browser Cache

If experiencing issues:

1. Clear browser cache/cookies
2. Return to Regula
3. Data remains (stored on server)

### Timezone

Regula uses your browser's timezone:

- Dates are shown in your local timezone
- Times are converted automatically
- Adjust in browser settings if needed

## Troubleshooting

### Settings Not Saving?

- Check for errors in form
- Ensure valid values (numbers for weeks, etc.)
- Click Save again
- Refresh page to verify

### Export Not Working?

- Browser may have blocked download
- Check browser's download restrictions
- Try different format (JSON vs CSV)
- Try different collection

### Ahead Weeks Not Updating?

- Refresh dashboard page
- May need up to 1 minute
- Check setting was actually saved
