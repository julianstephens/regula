# Modules & Lessons

Modules and lessons are where you organize the actual study content. Learn how to create, manage, and use them effectively.

## Modules Overview

A **Module** is a section of a program containing related lessons. Think of modules as chapters or units.

### Module Properties

- **Title**: Name (e.g., "Chapter 3: Algorithms")
- **Program**: Which program it belongs to
- **Start Date**: When the module begins
- **End Date**: When it ends

### Why Use Modules?

Modules help:

- Organize lessons into logical groups
- Track progress by section
- Filter lessons on program detail page
- Structure complex programs
- Schedule work by week or topic

## Creating Modules

### Via the Modules Page

1. Go to **Modules** in the sidebar
2. Click **New Module**
3. Fill in:
   - **Title**: Module name (required)
   - **Program**: Select which program (required)
   - **Start Date**: When module starts (required)
   - **End Date**: When module ends (required)
4. Click **Create**

### Via Program Detail

1. Open a **Program**
2. Click the **Modules** tab
3. Lessons within that program appear
4. Use "New Module" button at the top
5. Create the module (defaults to current program)

### Best Practices

- Keep module titles clear and specific
- Set realistic date ranges based on content
- Make start/end dates align with lessons inside
- One module per chapter/unit (usually weekly or topical)

## Filtering Modules

On the **Modules** page:

1. Use the **Program** filter at the top
2. Select a program to see only its modules
3. Select "All programs" to see everything

**Use case**: Focus on one program at a time.

## Viewing Module Details

Click a module title to open its detail page:

- Module name and dates
- Associated program
- List of lessons in the module
- Lesson count and status summary
- Quick actions (edit, delete)

## Editing Modules

1. Go to **Modules**
2. Click on the module to open it
3. Click **Edit**
4. Update fields:
   - Title
   - Program
   - Start/end dates
5. Click **Update**

**Impact**: Changes the container but doesn't affect lessons directly.

## Deleting Modules

**Warning**: Deletes the module but not individual lessons inside.

1. Open the module detail
2. Click **Delete**
3. Confirm

**After deleting:**

- Lessons remain but are no longer grouped
- They still belong to the program
- Filter by module disappears from program view

## Lessons Overview

A **Lesson** is an individual learning unit. Each lesson represents specific material you need to learn.

### Lesson Properties

- **Title**: What you're learning (required)
- **Program**: Which program it belongs to (required)
- **Module**: Which module (optional)
- **Due Date**: When it must be done (required)
- **Available Date**: When you can start (optional)
- **Type**: Category like "reading", "problem set", etc. (optional)
- **Course**: Associated course (optional)
- **Status**: Not Started, Active, Blocked, or Completed
- **Content**: Links, notes, or descriptions (optional)

## Creating Lessons

### Via Program Detail

1. Open a Program
2. Click the **Lessons** tab
3. Click **New Lesson**
4. Fill in the form

### Via Module Detail

1. Open a Module
2. Click **New Lesson**
3. Fill in the form (module is pre-selected)

### Via Lessons in a Program

Some programs allow creating lessons directly. Fill the form:

**Required Fields:**

- **Title**: What the lesson is about
- **Program**: Which program
- **Due Date**: When it's due

**Optional Fields:**

- **Module**: Group with other lessons
- **Available Date**: When you can start
- **Type**: Category (reading, assignment, etc.)
- **Content**: Links, notes, or descriptions

## Lesson Status

Each lesson has a status that changes as you work:

| Status          | Meaning               | Can Change To             |
| --------------- | --------------------- | ------------------------- |
| **Not Started** | Haven't begun         | Active, Blocked           |
| **Active**      | Currently working     | Completed, Blocked        |
| **Blocked**     | Depends on other work | Not Started               |
| **Completed**   | Finished              | Active (revise if needed) |

### Changing Status

In the lesson detail view:

- Use the **Status** dropdown to change
- Select new status from options
- View timeline of status changes

## Editing Lessons

1. Go to program or module containing the lesson
2. Click on the lesson title
3. Click **Edit**
4. Update fields
5. Click **Update**

**Frequently changed:**

- Due dates (reschedule)
- Status (progress tracking)
- Content (add resources)

## Deleting Lessons

**Note**: This is permanent.

1. Open the lesson detail
2. Scroll to bottom
3. Click **Delete**
4. Confirm

Deleting removes:

- The lesson itself
- Associated sessions record
- Timeline events
- But leaves reviews for reference

## Viewing Lesson Detail

Click a lesson to open its detail page with tabs:

### Overview Tab

Shows:

- Lesson title, module, program
- Due date and available date
- Current status
- Content/notes
- Edit button

### Sessions Tab

Shows all **Study Sessions** logged for this lesson:

- Date of session
- Duration in minutes
- Notes
- Total time spent

**Log a session:**

- Use built-in timer
- Or manually log time

### Events Tab

Chronological timeline of changes:

- Status changes
- Date modifications
- Other significant events

**Use it to:**

- See when you started (first status change)
- Track progress over time
- Understand when things got blocked

### Reviews Tab

Shows spaced-repetition reviews for this lesson:

- Review due dates
- Review status
- Timeline of reviews

**Start a review:**

- Click a review to start
- Update your understanding
- Regula schedules next review

## Tracking Progress

### Sessions and Time

Log how long you spend on each lesson:

**Using the Timer:**

1. Open the lesson
2. Click **Start Timer** button
3. Click **Stop** when done
4. Optionally add notes
5. Session is logged automatically

**Manual Entry:**

1. Open the lesson
2. Click **Log Session**
3. Enter duration in minutes
4. Enter date (defaults to today)
5. Add optional notes
6. Click **Log**

### Time Summary

In the lesson's **Sessions** tab:

- See all logged sessions
- Total time spent shows at top
- Helps track effort vs. outcome

## Lesson Organization Tips

### Scheduling Lessons

- Set **Available Date** when you can start
- Set **Due Date** for when it's due
- Dashboard shows lessons within your "Ahead Weeks"
- Plan so lessons don't overlap too much

### Creating in Batches

For a multi-week module:

1. Create the module first with full date range
2. Break into lessons (one per chapter, topic, etc.)
3. Space due dates across module duration
4. Leave dates that align with reading schedule

### Different Lesson Types

Use the **Type** field to categorize:

- Reading + notes
- Problem set
- Quiz
- Project
- Lab
- Discussion
- Essay

Helps organize and filter later.

### Content Links

In the **Content** field, include:

- Links to textbooks or materials
- Notes you wrote
- Relevant external resources
- Questions to remember

Use markdown formatting for readability.

## Tips for Success

### Start with Modules

Create modules for each chapter or unit first, then populate with lessons. This ensures dates align naturally.

### Set Realistic Dates

Leave buffer time between due dates. Don't pack everything into too short a time.

### Use Status Consistently

- Mark **Active** when you start work
- Mark **Completed** immediately after submission
- Mark **Blocked** if dependent on another lesson

This keeps your dashboard accurate.

### Log Sessions Regularly

The more you log:

- Better analytics
- More accurate progress tracking
- Better personal insights

Even rough estimates help.

### Review Regularly

Don't skip reviews. They reinforce learning through spaced repetition.

### Leverage Calendar View

Use the **Calendar** to see all lessons by date. Helps visualize workload across weeks.
