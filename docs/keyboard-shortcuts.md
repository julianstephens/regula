# Keyboard Shortcuts

Speed up your workflow with keyboard shortcuts. More coming soon!

## Global Shortcuts

Currently, Regula relies on standard browser shortcuts. Here are useful general shortcuts:

### Browser Navigation

| Action       | Shortcut             |
| ------------ | -------------------- |
| Go back      | Alt + ← or Backspace |
| Go forward   | Alt + →              |
| Refresh page | F5 or Ctrl+R         |
| Hard refresh | Ctrl+F5              |
| New tab      | Ctrl+T               |
| Close tab    | Ctrl+W               |

### Text Editing

| Action     | Shortcut |
| ---------- | -------- |
| Select all | Ctrl+A   |
| Copy       | Ctrl+C   |
| Paste      | Ctrl+V   |
| Cut        | Ctrl+X   |
| Undo       | Ctrl+Z   |
| Redo       | Ctrl+Y   |

## Tips While Waiting

### Use Browser Find

- Press Ctrl+F to open search
- Find lesson names in tables
- Find program names quickly

### Tab Navigation

- Use Tab to move between form fields
- Enter submits many forms
- Escape closes dialogs/modals

## Accessibility

Regula is designed to be accessible:

- Full keyboard navigation support
- Screen reader friendly
- High contrast options
- Clear focus indicators
